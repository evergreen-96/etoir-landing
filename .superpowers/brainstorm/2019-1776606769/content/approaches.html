Три стратегии — какой уровень изменений?

Выбери подход

A

Хирургический (2–3 часа)

Убираем симптомы AI-дизайна, не трогая структуру.

  • Убрать бейджи из 6 из 8 секций — оставить только Hero и CTA
  • Убрать иконки-кружки из feature-карточек, оставить только текст
  • Добавить 1–2 «сбивки ритма»: нечётное число колонок, полноширинный элемент
  • Цвет и copy не трогаем

Результат: −60% "AI-feel", структура та же

B

Design Refresh (полдня)

Меняем визуальный язык страницы целиком.

  • Всё из варианта A
  • Новая палитра: уходим от монотонного синего, добавляем тёплый акцент (amber / slate)
  • Асимметричные секции: чередование текст-лево / текст-право вместо центра
  • Разная плотность: одни секции сжатые, другие с большим воздухом
  • Заголовки: даём рекомендации по переписке (конкретика вместо шаблонов)

Результат: −90% "AI-feel", новый характер страницы

C

Полный редизайн (несколько дней)

Начинаем с визуального характера, не с текущего HTML.

  • Всё из варианта B
  • Новая структура секций (другой порядок, другие пропорции)
  • Референсный стиль: Linear, Vercel, Raycast — выбираем направление
  • Полная перепись заголовков и подзаголовков
  • Переработка Hero-секции как главного «лица» продукта

Результат: новый лендинг с узнаваемым характером

Я рекомендую B — Design Refresh: даёт максимальный эффект при разумных трудозатратах. Но скажи что выбираешь — покажу конкретные mockup-ы.
