## Турбо-страницы (Step 11 — частично)

**Сделано:** базовый RSS 2.0 канал `/rss.xml` со ссылками на 3 статьи блога. Подходит для индексации Яндексом и стандартных RSS-читалок.

**Что осталось для полноценного Turbo:**
1. Расширить RSS — внутрь каждого `<item>` добавить `<turbo:content>` с разрешённой HTML-разметкой статьи (h2-h6, p, ul/ol, img, blockquote и т.п.).
2. Подключить фид в [Яндекс.Вебмастер](https://webmaster.yandex.ru/) → раздел «Турбо-страницы для контентных сайтов».
3. Настроить кнопки и блоки (контакты, реклама, метрика) в шаблоне Турбо.

Полноценная разметка требует перекладки текста каждой статьи из HTML в `<turbo:content>` блок RSS — это разовая работа, делается отдельно.

## Действия в Яндекс.Вебмастер (Step 12)

### Обязательное (одноразово):

1. **Добавить и подтвердить оба домена** в [webmaster.yandex.ru](https://webmaster.yandex.ru/):
   - `etoir.ru` (основной)
   - `e-toir.ru` (зеркало)
   - Тег `yandex-verification` уже стоит на главной (`c62ee75976c65372`).
2. **Главное зеркало:** Настройки → Переезд сайта → указать `etoir.ru` основным. Подождать 2–4 недели на склейку.
3. **Регион:** Информация о сайте → Региональность → присвоить «Россия» (или конкретный город, если локальный бизнес). Влияет на ранжирование в Яндексе.
4. **Sitemap:** Индексирование → Файлы sitemap → добавить `https://etoir.ru/sitemap.xml`.
5. **RSS / Турбо:** Турбо-страницы → Источники → добавить `https://etoir.ru/rss.xml` (после расширения фида до полного Turbo-формата, см. выше).
6. **Связать с Метрикой:** Настройки → Привязка счётчика → выбрать ID счётчика Метрики (после Step 2).

### Рекомендуемое:

7. **Добавить организацию в [Яндекс.Бизнес](https://business.yandex.ru/):**
   - Карточка компании + телефон + рабочие часы → колдунщик в выдаче.
   - При наличии офиса — точка на Картах.
8. **Подать на индексацию вручную** после деплоя правок: Индексирование → Переобход страниц → вставить главную и блог-страницы.
9. **Мониторинг:**
   - ИКС (Индекс Качества Сайта) — обновляется раз в 1–2 месяца.
   - Поисковые запросы → отслеживать рост позиций по «CMMS система», «ТОиР программа», «автоматизация ТОиР».
   - Диагностика → еженедельно смотреть «фатальные» и «критичные» ошибки.
10. **Search Console (Google)** — параллельно: [search.google.com/search-console](https://search.google.com/search-console). Добавить домен через DNS-верификацию.
